### Животные (Базовый класс)
- **Имя**: строка (например, "Том", "Шарик")
- **Возраст**: целое число (годы)
- **Вес**: число (кг)
- **Место обитания**: строка (например, "дом", "поля", "пустыня")
- **Рацион**: строка (например, "мясо", "растительная пища")

### 1) Домашние
#### a) Собака
- **Порода**: строка (например, "Лабрадор", "Ши-тцу")
- **Размер**: строка (например, "маленькая", "средняя", "большая")
- **Уровень активности**: строка (например, "высокий", "умеренный", "низкий")
- **Обучаемость**: строка (например, "легкая", "умеренная", "сложная")

#### b) Кот
- **Порода**: строка (например, "Сиамский", "Персидский")
- **Степень независимости**: строка (например, "высокая", "умеренная", "низкая")
- **Любимая игра**: строка (например, "мягкая игрушка", "мышь")
- **Общительность**: строка (например, "социальный", "осторожный")

#### c) Хомяк
- **Вид**: строка (например, "Золотистый", "Дворцовый")
- **Активность в ночное время**: булевый (например, True, если активен ночью)
- **Привычки**: строка (например, "копать", "забираться")
- **Продолжительность жизни**: целое число (в месяцах)

### 2) Вьючные
#### a) Лошадь
- **Порода**: строка (например, "Арабская", "Теннесси")
- **Рост в холке**: число (см)
- **Минимальная нагрузка**: число (кг)
- **Скорость бега**: число (км/ч)

#### b) Верблюд
- **Количество горбов**: целое число (1 или 2)
- **Способность к выживанию в пустыне**: строка (например, "высокая", "умеренная", "низкая")
- **Питание**: строка (например, "растительность", "колючки")
- **Возраст размножения**: целое число (в годах)

#### c) Осел
- **Порода**: строка (например, "Иберийский", "Мул")
- **Выносливость**: строка (например, "высокая", "умеренная", "низкая")
- **Уровень слуха**: строка (например, "хороший", "обычный", "плохой")
- **Повадки**: строка (например, "спокойный", "упрямый")

### 1) Домашние (абстрактный класс)
- **Контакт с людьми**: строка (например, &quot;высокий&quot;, &quot;умеренный&quot;, &quot;низкий&quot;) — характеризует уровень взаимодействия с людьми.
- **Жизненная среда**: строка (например, &quot;дом&quot;, &quot;квартира&quot;) — описание места обитания.
- **Разнообразие рациона**: булевый (True/False) — указывает, могут ли они есть разнообразную пищу или нет.
- **Потребность в уходе**: строка (например, &quot;высокая&quot;, &quot;умеренная&quot;, &quot;низкая&quot;) — уровень ухаживания, который требуется.

### 2) Вьючные (абстрактный класс)
- **Мина́льная грузоподъемность**: число (кг) — минимальный вес, который животное может нести.
- **Послушание при работе**: строка (например, &quot;высокое&quot;, &quot;умеренное&quot;, &quot;низкое&quot;) — насколько животное обучаемо для работы под нагрузкой.
- **Способности к выживанию в трудных условиях**: строка (например, &quot;высокая&quot;, &quot;умеренная&quot;, &quot;низкая&quot;) — как животное адаптировано к работе в сложных условиях.
- **Долговечность и выносливость**: строка (например, &quot;высокая&quot;, &quot;умеренная&quot;, &quot;низкая&quot;) — способность животного работать долгое время.

from enum import Enum

class Species(Enum):
    СОБАКА = 'собака'
    КОТ = 'кот'
    ХОМЯК = 'хомяк'
    ЛОШАДЬ = 'лошадь'
    ВЕРБЛЮД = 'верблюд'
    ОСЕЛ = 'осел'

class Animal:
    def __init__(self, species: Species):
        self.species = species

# Примеры использования
animal = Animal(Species.СОБАКА)  # Верно
print(animal.species)

animal = Animal(Species.КОТ)  # Верно
print(animal.species)
